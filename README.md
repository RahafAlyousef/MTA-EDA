# EDA-Project
Welcome to MTA Turnstile Data Project
(MTA Traffic Analysis for Taxi Owners)

## Abstract:
Apply data analytics on the MTA subway data on traffic in stations.

Analyze turnstile data three months from the Jul – Sep 2021. 

## Tools:

• Technologies:
- Python.
- Jupyter Notebook.

• Libraries:
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Datetime
- sqlalchemy
